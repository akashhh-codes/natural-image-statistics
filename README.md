# Natural Image Statistics for Scene Classification

This project analyses statistical properties of natural images and applies handcrafted features for scene classification.

## Features Used
- Brightness
- Standard Deviation
- Edge Density
- Spatial Correlation

## Tasks
- Binary Classification (Botswana vs Indian Forest)
- Multi-class Scene Classification

## Tools
- Python
- OpenCV
- NumPy
- Scikit-learn
- Google Colab
